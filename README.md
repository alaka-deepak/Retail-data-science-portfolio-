
# Retail Data Science & Time Series Portfolio

This repository is a curated collection of end-to-end data science projects focused on retail and ecommerce business problems.

The portfolio covers:
- Demand forecasting
- Customer churn prediction
- Recommendation systems
- Inventory optimization
- Pricing analytics
- Customer segmentation
- NLP on product reviews
- Fraud detection
- MLOps and deployment

The goal of this repository is to demonstrate strong business understanding, machine learning expertise, and the ability to solve real-world retail challenges using data.
